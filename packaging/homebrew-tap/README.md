# Homebrew tap for Numan

This directory mirrors the formula published to the **`tonythethompson/homebrew-numan`** tap repository.

## Install

```bash
brew tap tonythethompson/numan
brew install numan
```

## Direct formula URL (no tap)

```bash
brew install --formula https://raw.githubusercontent.com/tonythethompson/numan/master/packaging/homebrew/numan.rb
```

## Maintainers

On each release, copy `packaging/homebrew/numan.rb` to `Formula/numan.rb` in the tap repo (or run `scripts/sync-homebrew-tap.sh`).
